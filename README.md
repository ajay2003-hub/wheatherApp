# Weather Application

A full-stack weather app using Node.js, Express, React, and OpenWeatherMap API.

## Features

- Search weather by city
- 5-day forecast
- Responsive design with Tailwind CSS
- Real-time weather data

## Setup

### Backend

```bash
cd server
npm install
# Add your OpenWeatherMap API key to .env
npm run dev
```

### Frontend

```bash
cd client
npm install
npm start
```
